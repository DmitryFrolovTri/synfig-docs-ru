# Импорт секвенций кадров

Чтобы импортировать секвенцию изображений, выберите "Файл" > "Импортировать последовательность", выберите все изображения секвенции и нажмите "Загрузить". Обработка больших секвенций может занять время. Импортированная секвенция появится в виде [слоя-переключателя](../instrumenty-animacii/sloi-pereklyuchatel) на панели слоев.
